# Flutter - Paw Paw

///////

## Run Locally

Clone the project

```bash
  git clone https://github.com/mikailkse/pawpaw.git
```

Go to the project directory

```bash
  cd pawpaw
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
